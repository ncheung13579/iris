IRIS GPT-2 Small Archive
========================

This directory contains all results, checkpoints, figures, and metrics
generated while IRIS used GPT-2 Small (124M params, 12 layers, d_model=768)
as its security sensor model.

Archived: 2026-03-14
Reason:   Upgraded to GPT-2 Large (774M params, 36 layers, d_model=1280)
          based on mimicry diagnostic results (notebook 20) showing GPT-2
          Large resolves the 100% mimicry evasion problem.

Contents:
  notebooks/    - Colab results notebooks (*_results.ipynb) with outputs
  results/      - Figures (PNG) and metrics (JSON) from experiments
  checkpoints/  - Trained SAE weights, activation caches, sensitivity scores

Key parameters (GPT-2 Small era):
  Model:           gpt2 (124M)
  d_model:         768
  SAE d_input:     768
  SAE d_sae:       6144 (8x) or 3072 (4x)
  Target layer:    0
  Sparsity coeff:  1e-4

These files are preserved for reference but are NOT used by the current
codebase, which now targets GPT-2 Large with d_input=1280, d_sae=10240,
and target layer 35.
